Recommended courses

There are many outstanding web design and development related courses in the lynda.com Online Training Library. So many, in fact, that it can be hard to know exactly what to take next. Below is a list of courses I recommend for new web designers.

Web Technology Fundamentals

Companion course to Web Design Fundamentals. This course focuses on the terms and technology that drives the web. This course is your "plain English" guide to the world of domains, databases, servers, and the technology that makes the web work.

HTML Essential Training

This course is designed to introduce web designers to the basics of HTML. Course focuses on basic syntax, common elements, and best practices for writing HTML.

Managing a Hosted Website

Shows how to set up a web site on a hosted server and perform common server tasks. The course also explains setting up password protection, creating a MySQL database, and working with subdomains.

CSS Fundamentals

Gives a high-level overview of Cascading Style Sheets and explores the basic concepts, terminology, and tools of the language. This course is perfect for people who want a big-picture overview before taking hands-on courses.

CSS: Core Concepts

In this hands-on course, you'll explore the foundational concepts of CSS as you learn syntax and best practices for coding CSS. You'll also explore the tools needed to work with CSS, how to properly apply CSS, and how to resolve conflicts.

CSS: Page Layouts

This course introduces basic layout concepts, shows how to create properly structured HTML based on prototypes and mockups, and goes into critical page layout skills such as floats and positioning.

Foundations of UX: Accessibility

In this course, you'll learn how accessibility helps your sites by providing equal access and opportunity to people with a diverse range of abilities. This comprehensive overview introduces the core concepts of accessibility and includes a look at the assistive technology visitors may use to access your projects.

JavaScript Essential Training

Introduces you to JavaScript while focusing on the current best practices and practical uses. You'll covering syntax, working with the DOM, and developing and debugging across multiple platforms, devices, and browsers.

JavaScript: Functions

You truly master the use of JavaScript once you learn how to work with functions. In this course, you'll explore how functions work, how they're declared, and how they're invoked, and then complete challenge exercises that allows you to test your knowledge.
