Specifications

Perhaps the best way to learn the languages necessary for web design is to read their actual specifications. These specs are used by browser manufacturers and other user agents as guides when implementing the languages. The can be difficult to read at first, but are a wealth of information.

CSS Current Work

Central location for tracking the status of all the modules of the CSS specifications. You'll find progress reports, an overview of the current state of CSS, and links to all the specs.

HTML5 (W3C)

The W3C version of the HTML5 specification. It diverges some from the WHATWG version as it seeks to define a stable implementation.

HTML (WHATWG)

The WHATWG version of the HTML5 specification. It is considered a 'living' spec and therefore doesn't have a version number.

ECMA 262 Specification

Defines the ECMA scripting standard, which JavaScript is built off of.

WAI-ARIA

Specification that outlines the creation of accessible web applications. Helpful for learning how to create accessible sites.

All W3C Specifications

Huge list of all W3C Specifications. Great place to research the standardization process.
